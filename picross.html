Steampunk Picross

Steampunk Picross

  • Picross game I made on my own!

This is a game I made on my own, outside of work. It is a picross game with procedurally generated puzzles. There is a short "story" mode that is steampunk themed.

All of the animations and systems were made from scratch. Most of the art was pulled from google images, so I never published the game.

You can download the Android APK and a ZIP of the C# code below!

  • Created procedural grid generation mode with over 2.5 x 10120 possible puzzles
  • Developed story mode with scalable amount of puzzles
  • Programmed system that automatically converts pixel-art into picross puzzles
