Draw a Stickman: EPIC 2 AR Effect

Draw a Stickman: EPIC 2 AR Effect

  • AR effect that features interactable characters from the “Draw a Stickman” games!

This project is an AR Effect created with the Sony Xperia package in Unity. It is based in the Draw a Stickman franchise by Hitcents, and the app is an Xperia device exclusive.

I handled everything in the project, scripting new interactions that work well in the AR environment. Most assets and scripts were pulled from "Draw a Stickman: EPIC 2".

This AR Effect includes both a surface effect with animated characters that the player can move around and interact with, as well as face filters that act as masks of NPCs from EPIC 2.

  • Created AR interactable versions of characters and objects from the franchise
  • Implemented NPC face filters using Unity-Xperia environment
