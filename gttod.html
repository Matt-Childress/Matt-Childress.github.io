Get to the Orange Door

Get to the Orange Door, 2018 - 2020

  • Rogue-lite FPS with parkour abilities and a retro style

Get to the Orange Door is a game in which I had to jump into a large, pre-existing project in order to handle its first time being ported to consoles from PC.

My tasks in this project included implementing console controls and logic for Nintendo Switch playmodes, as well as updating many existing plugins and SDKs.

I lead the initial efforts in porting the game to consoles.

  • Lead developer in porting the game to consoles through Unity
  • Refactored existing systems to conform to Nintendo lotcheck guidelines
  • Implemented console controls, update existing plugins
