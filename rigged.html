Rigged

Rigged, 2016

  • Gerrymandering puzzle game in conjunction with a documentary featured on Netflix

Rigged is a 2D game that has a lot of grid logic and code that handles automatically resizing "districts" within the grid.

Hitcents partnered with Fusion to make the game based on a political strategy of reshaping districts. I was the only programmer that worked on this project and I worked with an in-house team of designers and artists.

My tasks for this project included making a level editor for designers, handling the core game mechanics of district resizing through touch input, and implementing in-app-purchasing and ads.

  • Created level editor used in-house and designed game levels
  • Programmed all game logic and handled Android and iOS publishing
  • Compiled assets in Unity throughout development and documentary filming
