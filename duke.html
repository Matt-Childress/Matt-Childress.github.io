Duke of Defense

Duke of Defense, 2019 - 2020

  • Action-based tower defense game in which the player turns trees into defenses

My role in this project was to introduce the external developers to the technologies used in porting the project to Nintendo Switch. I guided the porting process by providing samples of how to set up systems such as saving and loading, as well as debbuging issues with their operating environment and builds.

  • Guided process of porting the game to consoles
  • Set up third-party plugins versions that were supported by the Nintendo Switch
  • Solved build issues
