A Robot Named Fight!

A Robot Named Fight!

  • Roguelike Metroidvania focused on exploration and item collection!

This is an indie game that was not developed by Hitcents. Instead Hitcents published the game on certain platforms.

My role in this project was getting the developer started with Nintendo Switch porting, and guiding the porting process from that point.

I provided assistance with updating to the Nintendo-supported version of the Rewired plugin, and wrote code for the developer that handles player saves with the Nintendo Switch filesystem.

  • Provided sample code for systems such as player saves
  • Assisted in updating third party plugins
  • Set up tests for lotcheck submission
